# VPN - OpenConnect

# Install
store the project into `/home/<username>/.local/share/gnome-shell/extensions/vpn@openconnect.com`

Essential folder structure:
```
 - vpn@openconnect.com
    - img
     - icon_connected.svg
     - icon_disconnected.svg
    - extension.js
    - metadata.json
    - stylesheet.css
```
